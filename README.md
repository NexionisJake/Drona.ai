Anti-Copilot is a **Socratic Code Editor** designed to prevent "Vibe Coding" ‚Äî the habit of blindly pasting AI-generated code without understanding it. 

When you paste a large block of code (e.g., >25 lines), the editor **LOCKS**. To unlock it, you must answer Socratic questions generated by an AI Mentor about the code you just pasted. Prove you understand it, or delete it and write it yourself.

## üöÄ Features

- **Paste Trap**: Detects large pastes and instantly locks the editor.
- **Socratic Mentor**: Uses **Anthropic Claude 3.5 Sonnet** to generate context-aware questions about the pasted code's logic, security, and performance.
- **Builder Score**: Gamified scoring system that rewards manual typing (+0.2/burst) and penalizes pasting (-0.2/line).
- **Escape Hatch**: Press `Ctrl+Z` to "Reject the Vibe", undo the paste, and refund your score penalty.
- **Vibe UI**: A "GitHub Dark" aesthetic with glassmorphism, animations, and a dedicated Mentor Sidebar.

## üì¶ Setup & Run

### Prerequisites
- Node.js v18+
- Python 3.11+
- An Anthropic API Key (`ANTHROPIC_API_KEY`)

### Quick Start

1.  **Clone the repo**
2.  **Set up Environment Variables**:
    ```bash
    cp .env.example .env
    # Edit .env and add your ANTHROPIC_API_KEY
    ```
3.  **Run the Full Stack**:
    ```bash
    ./run.sh
    ```
    This script sets up the Python virtual environment, installs dependencies, and launches both frontend (`localhost:5173`) and backend (`localhost:8000`).
